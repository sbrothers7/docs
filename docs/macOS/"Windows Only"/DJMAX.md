# DJMAX

## Installation

!!! Note
    Crossover requires a paid subscription

1. Install Crossover
`brew install --cask crossover`
2. [Install Steam on Sikarugir](https://github.com/sbrothers7/docs/wiki/MacOS-Sikarugir#installing-steam-with-sikarugir)
3. Download DJMAX
4. Open Crossover
5. Click *Install*
6. Select Steam, then Click *Install*
7. Go through the installation process
8. Go to Bottles > Bottle Actions > Open C: Drive
9. Open Steam folder in sikarugir (found in `~/Applications/Sikarugir/yourWrapperName.app/Contents/SharedSupport/prefix/drive_c/Program Files (x86)/Steam`)
10. Replace the Steam folder for Crossover with the sikarugir folder
11. Run steam in Crossover
12. Open DJMAX
