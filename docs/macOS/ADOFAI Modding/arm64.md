# ADOFAI Modding (arm64)

## Using kkorenn/unity-mod-manager

[Download the executable](https://github.com/kkorenn/unity-mod-manager/releases/latest) and use the GUI.

OR

Download my [script](https://github.com/sbrothers7/UMMInstall).

## Using MelonLoader + UMMCompat

[Download the modlist.org app](https://github.com/modlist-org/modlist_org_app) to download UMMCompat. This will automatically download and install the latest version of MelonLoader.

OR

Manually download and install [MelonLoader](https://melon-loader.com/) and download the [UMMCompat plugin](https://github.com/modlist-org/UMMCompat/releases/latest) and put it at `<game dir>/Plugins`

!!! note
    Make sure to set steam launch options to:
    `"~/Library/Application Support/Steam/steamapps/common/A Dance of Fire and Ice/setup_helper.sh" %command%`
